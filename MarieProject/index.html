MARIE CODE MADE SIMPLE

This page is an attempt to illustrate the use of MARIE commands to solve simple problems,with the hope of aiding the better understanding of MARIE code for the reader . I hope I save you time , so you can spend it understanding concepts that will enlighten you and in turn the rest of the world.

Here are some commands and descriptions of what they do

  • INPUT-prompt user input
  • OUTPUT-show what is in AC on output screen
  • LOAD-loads specified element into AC
  • STORE-save element in AC to specified location
  • SKIPCOND-jumps next instruction on specified condition
